DIY-DRONE

April 2018

Ever wanted to build a drone by yourself?

We will make that dream come true!

1. Getting Started

1.1 Hardware

First, you'll need all the parts in order to assemble a drone yourself. You can purchase a kit from our shop, which includes everything you need (and helps us develop new projects). If you want a bigger challenge, you can also find all the parts online, but you'll need to build the PCB and enclosure yourself.

While you're eagerly waiting for you package to arrive, you should read through this article, so you'll already know what's coming next.

1.2 GitHub

All files for this project can be found on GitHub (github.com/cloud-garage/diy-drone). You will have to download or clone the repository to your PC to get started.

The folder structure is pretty self-explenatory, but you can still read about it in the README.md. Each directory also has its own readme file with all necessery explanations. You can play around with the project to become familiar with it.

1.3 Questions & Help

If you have any questions or problems, feel free to comment on the bottom of this page and we'll look into your problem.
